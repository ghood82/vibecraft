Enter GSD (Get Stuff Done) mode. Plan, prioritize, and execute.

Read the orchestrator skill at `${CLAUDE_PLUGIN_ROOT}/skills/orchestrator/SKILL.md` for GSD workflow.

**If starting a new plan:**
1. Understand the goal: $ARGUMENTS
2. Define "done" — what does success look like?
3. Break into tasks with clear completion criteria
4. Prioritize by impact vs effort
5. Create a TodoList and start executing

**If continuing existing work:**
1. Check CLAUDE.md and any existing task lists
2. Review what's been done and what's remaining
3. Re-prioritize if needed
4. Continue executing the next highest-priority task

**Principles:**
- Ship the highest-leverage thing first
- One thing at a time, done well
- If blocked, switch to the next task — don't sit idle
- Review after each task: did it meet the bar?
